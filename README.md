# Avances Territoriales
Esta aplicación fue adaptada por el grupo Semillero de Innovación Geográfica (SIG) de Esri Colombia. Para mayor información consulte www.esri.co o visite @geo_geeks en twitter.
Esta aplicación permite visualizar avances territoriales de diferentes sectores a nivel nacional y compararlos con distintos indicadores base de población.

## Instrucciones 

El menú de filtros le permite seleccionar tanto los indicadores base como los indicadores según el sector en cada departamento.

El menú leyenda mostrará las convensiones usadas para representar los indicadores base y el rango de los datos representados.
